# Overview

Chips is **invisible commerce infrastructure** for the agentic web. Three layers compose into one network:

1. **Hosted A2A enterprise agent runtime** — multi-tenant, BYO-LLM, eleven JSON-RPC methods over HTTP + SSE
2. **Cross-enterprise context bridge** — consent-mediated context queries across the network, scope vocabulary governed by the Linux Foundation A2A WG
3. **Network learning** — five loops, two levels, Level B under differential privacy

This Overview is a placeholder. The full docs land alongside the Phase 0 runtime.
